# E2E Integration (Template)

## Bandas de ejecucion

- `test:e2e:quick`: iteracion local rapida.
- `test:e2e:ci`: smoke API + web publica + CLI.
- `test:e2e:full`: suite completa con auth web.

## Variables base sugeridas

- `E2E_BASE_URL`
- `E2E_PORT`
- `E2E_API_KEY`
- `E2E_ADMIN_API_KEY`
- `E2E_USER_API_KEY`
- `E2E_TECHLEAD_API_KEY`
- `E2E_DAILY_PROJECT_ID`
- `E2E_NON_MEMBER_PROJECT_ID`
- `E2E_AI_PROJECT_ID`
- `E2E_AS_USER_EMAIL`
- `E2E_CLERK_TEST_EMAIL` (solo full)
